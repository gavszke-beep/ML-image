# Нейронная сеть для распознавания рукописных цифр

Простая нейронная сеть с прямой связью (feedforward) для распознавания рукописных цифр из датасета MNIST. Реализована на чистом NumPy без использования готовых фреймворков глубокого обучения.

## 📋 Описание

Этот проект демонстрирует реализацию базовой нейронной сети с одним скрытым слоем для классификации изображений рукописных цифр (0-9). Сеть обучается на датасете MNIST и может делать предсказания на пользовательских изображениях.

### Архитектура сети
- **Входной слой**: 784 нейрона (28×28 пикселей)
- **Скрытый слой**: 20 нейронов с сигмоидной функцией активации
- **Выходной слой**: 10 нейронов (по одному на каждую цифру) с сигмоидной функцией активации

## 🚀 Возможности

- Обучение на датасете MNIST
- Оценка качества модели (функция потерь и точность)
- Классификация пользовательских изображений
- Визуализация результатов
